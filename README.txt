Como usar

1. Copie este arquivo para:
   static/js/house-gallery-data.js

2. Coloque as imagens dentro de:
   static/img/casa/

3. Os nomes das imagens precisam ser exatamente:
   ambiente1.jfif
   ambiente2.jfif
   ambiente3.jfif
   ambiente4.jfif
   ambiente5.jfif
   ambiente6.jfif
   ambiente7.jfif
   ambiente8.jfif
   ambiente9.jfif
   ambiente10.jfif

4. Depois faça commit e push para subir ao servidor.

Se alguma imagem não aparecer, revise:
- nome do arquivo
- extensão .jfif
- letras maiúsculas/minúsculas
- caminho da pasta
